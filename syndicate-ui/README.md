# Syndicate UI

React 19 + Vite + TypeScript + Tailwind v4 frontend for the Syndicate developer orchestration platform.

## Run Locally

```bash
npm install
npm run dev
```

Opens at http://localhost:5173

## Pages

| Route | Page | Description |
|-------|------|-------------|
| / | Dashboard | Overview metrics, getting started |
| /live | Live Room | Real-time agent collaboration feed |
| /agents | Agent Roster | Agent status, roles, models |
| /tasks | Task Pipeline | Kanban-style task lifecycle |
| /memory | Memory | Accumulated learnings and skill evolution |
